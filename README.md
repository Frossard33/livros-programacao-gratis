# 📚 Biblioteca de Programação Gratuita

Bem-vindo à **Biblioteca de Programação Gratuita**.

Este repositório reúne diversos **livros, apostilas e materiais de estudo de programação** para ajudar estudantes que querem aprender tecnologia mas não têm acesso fácil a materiais pagos.

O objetivo é criar uma **biblioteca aberta de conhecimento** para iniciantes e desenvolvedores que desejam evoluir na área de tecnologia.

---

# 🚀 Objetivo do Projeto

A área de tecnologia possui muito conhecimento espalhado pela internet, mas muitas vezes é difícil para iniciantes encontrar materiais organizados.

Este repositório foi criado para:

* 📚 reunir livros de programação
* 🧠 ajudar iniciantes a aprender tecnologia
* 💻 facilitar o acesso ao conhecimento
* 🚀 incentivar pessoas a entrarem na área de desenvolvimento

# 🧠 Sugestão de trilha de estudos

Se você está começando agora na programação, recomendamos a seguinte ordem de estudo:

1️⃣ Lógica de programação
2️⃣ Algoritmos
3️⃣ Python ou JavaScript
4️⃣ HTML e CSS
5️⃣ Banco de dados (SQL)
6️⃣ Desenvolvimento Web
7️⃣ Estruturas de dados
8️⃣ Engenharia de software
